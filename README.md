# UK Housing Data Analysis using R Language and R Studio
This project explores the UK housing market using real-world property price data to understand the key factors that influence house prices. The analysis is performed using R programming and focuses on data cleaning, exploratory data analysis (EDA), and statistical modelling.

The goal is to transform raw housing data into meaningful business insights that can support better decision-making in real estate and investment contexts.

## Objectives

- Clean and prepare UK housing data for analysis
- Explore trends in house prices across regions and property types
- Identify key factors influencing property prices
- Build a regression model to quantify price drivers
- Present clear business insights from the data

## Key Analysis Performed

- Data cleaning and preprocessing (handling missing values, formatting, and duplicates)
- Exploratory Data Analysis (EDA) using visualisations
- Analysis of price distribution across regions and property types
- Time-based trend analysis of housing prices
- Multiple linear regression modelling to identify significant predictors

## Tools & Technologies

- R Programming
- tidyverse (dplyr, ggplot2)
- R Markdown
- Statistical Modelling (lm function)

## Data Visualisation

### Key Insights

- Housing prices vary significantly by region
- Property type has a strong impact on price differences
- Location remains the strongest predictor of house value
- Market trends show clear changes over time
